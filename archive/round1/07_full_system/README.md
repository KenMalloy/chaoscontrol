# Experiment 07: Full System

## Hypothesis
All components together outperform any subset — synergy test.

## Null hypothesis
full_system does not beat best_single.

## Predictions
- full_system closes the gap between vanilla_ssm and transformer
- Removing any single component (gate, Wernicke, memory) degrades performance
- full_system_medium and full_system_full show the advantage scales with size
- best_pair outperforms best_single (pairwise synergy exists)
- full_system outperforms best_pair (three-way+ synergy exists)

## Method
Ten configs: transformer ceiling, vanilla SSM floor, best single component, best pair,
three ablations (full minus one component each), and the full system at three sizes.
best_single and best_pair are generated by analysis/promote_winners.py after experiments 01-06.
model_dim=128 for small, 256 for medium, 384 for full. Fixed 300s budget.

## Dependencies
01-06 (all prior experiments must complete first).

## Kill criteria
If full_system does not beat best_single, the synergy thesis is wrong and components should be evaluated independently.
